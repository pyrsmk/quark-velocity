quark-velocity 0.1.3
====================

[Velocity](http://julian.com/research/velocity) module for [quark](https://github.com/pyrsmk/quark).

Install
-------

```shell
npm install quark-velocity
bower install quark-velocity
```

Use
---

```js
// Use it as usual on Velocity
$('.element').animate({
	left: 100,
}, {
	duration: 1000
});
```
